# Gestión de Estudiantes en Java

## Descripción

Este proyecto fue desarrollado como parte de las actividades de la materia de Programación Orientada a Objetos.

La aplicación simula un sistema sencillo de gestión de estudiantes utilizando los principales conceptos de POO en Java.

## Características

- Creación de clases y objetos.
- Uso de atributos y métodos.
- Implementación de constructores.
- Encapsulamiento.
- Herencia mediante clases derivadas.
- Sobrescritura de métodos (`@Override`).
- Reutilización de código.
- Aplicación de buenas prácticas y comentarios en el código.

## Tecnologías utilizadas

- Java 21
- IntelliJ IDEA
- Git
- GitHub
